The Story

Built from personal experience

BenzoTracker was originally built for my own tapering process. Over time, I realized it could help others facing the same challenges, so I made it publicly available, completely free.

  • ✓Clean, intuitive interface, no medical background needed
  • ✓Daily dose logging with timestamps, reasons, and comments
  • ✓Visual progress charts and trend analysis
  • ✓Cross-device access, desktop and mobile ready
  • ✓No ads, no subscriptions, no data selling, ever
  • ✓Ongoing development based on real user feedback
